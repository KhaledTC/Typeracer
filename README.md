# Typeracer
Practice typing quotes as fast as possible to improve your typing speed.

1. install NodeJs
2. run the following commands in the shell
   - npm i express
   - npm i ejs
   - npm i path
   - npm i axios
   - npm i nodemon
3. go to the project folder in the shell
4. run this command
   - nodemon index
5. go to http://localhost:5040/ in chrome and enjoy!
